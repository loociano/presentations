What is (are) Aria Templates?

  • Javascript framework
    • Client side template engine
  • For rich and scalable web apps
  • Created and maintained by Amadeus

Main features

  • Model-View-Controller Pattern in action
  • View functions (macros)
  • Inheritance
  • Bidirectional data bindings
  • Automatic refresh
  • Partial refresh
  • Improved DOM manipulation

AT vs jQuery

  • AT provides a full MVC framework
  • jQuery simplifies DOM access and manipulation
  • So can I use both?
    • Yes and no
    • They are not incompatible
    • Both provide utils to manipulate the DOM
