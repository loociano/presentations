What is (are) Aria Templates?

  • MVC Javascript framework
    • Client side template engine
  • For rich and scalable web apps
  • Created by Amadeus (2009)
  • Maintained since then
  • Open-sourced

AT star features

  • MVC Pattern in action
  • "View" functions (macros)
  • Javascript inheritance
  • Bidirectional data bindings
  • Partial refresh
  • Automatic refresh
  • Simplified DOM manipulation
  • CSS inheritance and programming
    • think sass
  • Wigdets
    • think Web Components, polymer

AT Maintenance

  • AT Team working agile
    • Located in Nice, France
  • Very frequent releases
    • AT 1.6.9 published December, 5
  • Active support and training

AT vs jQuery

  • AT is a full MVC framework
  • jQuery simplifies DOM access and manipulation
  • So can I use both together?
    • Yes and no
    • They are not incompatible
    • Both provide utils to manipulate the DOM
    • E-retail uses AT whenever possible

AT vs Angular

  • Angular by Google, is another MVC Javascript Framework
  • Webapps usually are built upon one framework
  • Angular has better performance, AT has to rely on cache
  • Angular more unstable, controversial changes from 1.x to 2.x
